# 安装Python

- [x] 从[Python官网](https://www.python.org/)下载Python安装包。
    - installer：完整版。[^1]
    - embeddable package：阉割版。
- [ ] 在cmd执行`python`：
    - [ ] 如果跳转到微软应用商店：
        - [ ] 设置 > 应用 > 高级应用设置 > 应用执行别名
            - [x] python.exe：`关`
            - [x] python3.ext：`关`
    - [x] 报错“'python'不是内部或外部命令，也不是可运行的程序或批处理文件”。[^2]

---

- [ ] Install Python 3.12.8 (64-bit)
    - [x] Use admin privileges when installing py.exe：`√`
    - [x] Add python.exe to PATH：`√`（添加环境变量）
    - [x] `Customize installation`
- [ ] Advanced Options
    - [x] Install Python 3.12 for all users：`√`

---

- [ ] 在cmd执行`python --version`：
    - [x] 打印`Python 3.12.8`
- [ ] 避免路径过长导致的错误：
    - [ ] 注册表编辑器 > `计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\FileSystem` > LongPathsEnabled > 修改
        - [x] 数值数据：`1`
    - [x] 重启电脑生效

[^1]: 包含完整的Python标准库、解释器（CPython）以及一些辅助工具（例如pip）等。

[^2]: 表示未安装Python或Python卸载干净。
