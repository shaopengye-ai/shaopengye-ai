# 为什么学习Python 05-06 视频day01 Python文献

- Python之父：龟叔（荷兰人 Guido van Rossum 吉多·范罗苏姆）
- Python的设计初衷：*介于C和Shell之间*，功能全面、易学易用、可拓展的语言。
    - C：全面调用计算机的功能接口 => 让计算机能更快运行 => 迫使程序员像计算机一样思考
    - Shell：丰富的命令 => 轻松编程
- Python的设计哲学：优雅、明确、简单。
- Python的优点：
    - 易于学习：相对较少的关键字，简单的结构，明确定义的语法。
    - 广泛的标准库：Python标准库提供了系统管理、网络通信、文本处理、数据库接口、图形系统、XML处理等额外的功能。
    - 大量的第三方模块。
        - => 适合用于：数据分析与科学计算、人工智能与机器学习。
    - 互动模式：可以从终端输入代码并立即获得结果。
    - 可移植：基于其开放源代码的特性，Python已经被移植到许多平台。
    - 可扩展：如果需要一段运行很快的关键代码，或者是想要编写一些不愿开放的算法，可以使用C或C++完成那部分程序，然后从Python程序中调用。
    - 免费、开源。
- Python的缺点：
    - 运行速度慢：和C相比非常慢，因为Python是解释型语言，代码在执行时会一行一行地翻译成CPU能理解的机器码，这个翻译过程非常耗时。而C是运行前直接编译成CPU能执行的机器码。
    - 代码不能加密：如果要发布Python程序，实际上就是发布源代码，这一点跟C不同，C不用发布源代码，只需要把编译后的机器码（也就是在Windows上常见的exe文件）发布出去。所以，凡是编译型的语言都没有这个问题，而解释型的语言则必须把源码发布出去。
        - Java发布.class字节码文件。
        - 大模型训练，做统计分析，无须发布。
