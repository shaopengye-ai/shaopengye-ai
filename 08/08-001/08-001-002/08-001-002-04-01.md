# 字符串 01-02 视频day04 Python文献

- 字符串 (String)：字符串不可变，元素为字符，元素可重复，元素有序。
- 创建字符串： #gemini-l
    - 普通字符串：`"str"`（优先）、`'str'`。
    - 文档字符串：`"""str"""`。
        - 类中第一行：类说明文档。
        - 函数中第一行：函数说明文档。
        - 多行注释。

```python
# hello 'world'
print("hello 'world'")

"""
hello
world
"""
print("""hello
world""")
```

- 访问字符串：同列表。

```python
s = 'hello world'
print(s[0], s[-1])  # h d
print(s[4:-3])  # o wo
```

- 字符串相加、字符串乘法：同列表。

```python
s1 = "hello"
s2 = "world"
s3 = s1 + s2
print(s3)  # helloworld
print(s3[0] is s1[0], s3[5] is s2[0], s3[4] is s2[1])  # True True True

s4 = s1 * 2
print(s4)  # hellohello
print(s4[0] is s1[0], s4[5] is s1[0])  # True True
```

- 检查子字符串是否在字符串中：同列表。

```python
s = "hello world"
print("lo" in s)  # True
```

- 原始字符串：没有转义，所见即所得。
    - 格式：`r"str"`

```python
print("hello\tworld")  # hello	world
print(r"hello\tworld")  # hello\tworld
```

- 常用函数：
    - *`s.replace(old, new[, count])`*：返回字符串副本，其中*子字符串`old`*被替换为`new`。
        - `count`：最大替换次数。默认为`-1`，表示不限制。
    - *`s.split([sep=None][, maxsplit=-1])`*：返回`s`中的单词列表，使用*字符串`sep`*拆分单词。
        - `sep`：默认为`None`，表示任何空白字符串，并从结果中丢弃空字符串。
        - `maxsplit`：最大拆分次数。默认为`-1`，表示不限制。
        - 单词个数 = 拆分次数 + 1
    - `s.rsplit([sep=None][, maxsplit=-1])`：类似`split()`，从右侧开始拆分。
        - 常用于获取文件路径的后缀或最后一段。
    - *`s.join(iterable)`*：串联`iterable`中的*字符串*。*字符串`s`*作为分隔符。
        - 大厂规范：严禁在循环内使用`+`连接字符串，必须使用`join()`。
    - `s.strip([chars])`：返回字符串副本，去除前导字符和尾随字符。
        - `chars`：指定要移除的*字符集*。默认为`None`，表示移除空白字符。
    - `s.lstrip([chars])`：类似`strip()`，仅去除前导字符。
    - `s.rstrip([chars])`：类似`strip()`，仅去除尾随字符。
    - `s.removeprefix(prefix)`：如果`s`以`prefix`开头，返回`s[len(prefix):]`；否则返回`s`。
        - Python3.9新增，比`strip()`更安全，不会误删字符集。
    - `s.removesuffix(suffix)`：如果`s`以`suffix`结尾，返回`s[:-len(suffix)]`；否则返回`s`。
        - Python3.9新增。
    - `s.upper()`：返回字符串副本，将每个字符转为大写。
    - `s.lower()`：返回字符串副本，将每个字符转为小写。
    - `s.casefold()`：返回字符串副本，将每个字符转为小写。
        - `lower()`只支持在Unicode中有明确小写对应形式的字符。
        - `casefold()`功能更强，还支持一些特殊的大小写映射关系，例如德语'ß'会转为'ss'。
    - `s.swapcase()`：返回字符串副本，将每个字符，大写转小写，小写转大写。
    - `s.capitalize()`：返回字符串副本，将第一个字符转为大写，其余字符转为小写。
    - `s.title()`：返回字符串副本，将每个单词的首字母转为大写，其余字母转为小写（标题格式）。
        - 单词：使用空白字符串分隔。
    - `len(s)`：返回字符串的长度。
    - `max(s)`：返回字符串中Unicode码点最大的字符。
    - `min(s)`：返回字符串中Unicode码点最小的字符。
    - *`s.find(sub[, start[, end]])`*：返回*子字符串`sub`*在`s[start:end]`中的最小索引。
        - 如果未找到，则返回`-1`。
    - `s.rfind(sub[, start[, end]])`：返回子字符串`sub`在`s[start:end]`中的最大索引。
        - 如果未找到，则返回`-1`。
    - *`s.index(sub[, start[, end]])`*：类似`find()`。
        - 如果未找到，则引发`ValueError`。
    - `s.rindex(sub[, start[, end]])`：类似`rfind()`。
        - 如果未找到，则引发`ValueError`。
    - `s.count(sub[, start[, end]])`：返回子字符串`sub`在`s[start:end]`中的个数。
    - `s.startswith(prefix[, start[, end]])`：如果`s[start:end]`以指定的`prefix`开头，返回`True`，否则返回`False`。
        - `prefix`：字符串或字符串元组。
    - `s.endswith(suffix[, start[, end]])`：如果`s[start:end]`以指定的`suffix`结尾，返回`True`，否则返回`False`。
        - `suffix`：字符串或字符串元组（检查多个后缀）。
    - `s.isspace()`：`s`是否非空，且只包含空白字符。
- 其他函数（了解）：
    - `s.center(width[, fillchar])`：返回字符串副本，长度为`width`，居中对齐。
        - `fillchar`：填充字符，默认为空格。
    - `s.ljust(width[, fillchar])`：类似`center()`，左对齐。
    - `s.rjust(width[, fillchar])`：类似`center()`，右对齐。
    - `s.zfill(width)`：类似`center()`，右对齐，使用`0`填充。
        - 对正负号有特殊处理，例如：`print("42".zfill(5), "-42".zfill(5))  # "00042" "-0042"`
    - `s.expandtabs(tabsize=8)`：将`s`中的所有`\t`替换为一个或多个空格，具体取决于当前列以及指定的`tabsize`。
    - `s.splitlines([keepends=False])`：返回`s`中的行列表，使用换行符（`\n`、`\r`、`\r\n`）拆分行。
        - `keepends`：是否保留换行符。
    - `s.partition(sep)`：在`sep`首次出现的位置拆分`s`，返回`(prefix, sep, suffix)`。
        - 如果未找到，返回`(s, "", "")`。
    - `s.rpartition(sep)`：类似`partition()`，从右侧开始拆分。
    - `s.encode(encoding="utf-8"[, errors="strict"])`：字符串`s` -> 字节序列`bytes`
        - `errors`：错误处理方案。
    - `s.format_map(dict)`：使用字典等映射关系数据来格式化字符串。 #todo
    - `s.maketrans(x, y[, z])`：为`translate()`创建转换表。 #todo
        - `tbl = s.maketrans("abc", "123", "d")  # a->1, b->2, c->3, 删除d`
    - `s.translate(table)`：根据给定的转换表替换或删除字符。
        - 批量替换或删除字符，性能最高的方式。
    - `s.isalnum()`：`s`是否非空，且只包含字母（英文字母+汉字）或数字。
    - `s.isalpha()`：`s`是否非空，且只包含字母（英文字母+汉字）。
    - `s.isascii()`：`s`是否只包含ASCII字符（空字符串是ASCII字符）。
    - `s.isdecimal()`：`s`是否非空，且只包含十进制数字。
    - `s.isdigit()`：`s`是否非空，且只包含数字（比`isdecimal()`多上标）。
    - `s.isnumeric()`：`s`是否非空，且只包含数值（比`isdigit()`多罗马数字和中文数字）。
    - `s.isidentifier()`：`s`是否为有效的Python标识符。
    - `s.isupper()`：`s`中是否包含至少一个区分大小写的字符，且所有这些（区分大小写的）字符都是大写。
    - `s.islower()`：`s`中是否包含至少一个区分大小写的字符，且所有这些（区分大小写的）字符都是小写。
    - `s.istitle()`：`s`是否非空，且符合标题格式（每个单词的首字母大写，其余字母小写）。
    - `s.isprintable()`：`s`是否只包含可打印字符（空字符串和空格是可打印字符）。
