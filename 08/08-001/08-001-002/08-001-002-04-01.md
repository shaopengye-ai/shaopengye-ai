- 字符串 (String)：元素有序、元素不可变、元素可重复。
- 定义字符串： #gemini-l
    - 普通字符串：`"str"`（优先）、`'str'`。
    - 文档字符串：`"""str"""`。

```python
# hello 'world'
print("hello 'world'")

"""
hello
world
"""
print("""hello
world""")
```

- 访问：同列表。

```python
s = 'hello world'
print(s[0], s[-1])  # h d
print(s[4:-3])  # o wo
```

- 字符串相加、字符串乘法：同列表。

```python
s1 = "hello"
s2 = "world"
s3 = s1 + s2
print(s3)  # helloworld
print(s3[0] is s1[0], s3[5] is s2[0], s3[4] is s2[1])  # True True True

s4 = s1 * 2
print(s4)  # hellohello
print(s4[0] is s1[0], s4[5] is s1[0])  # True True
```

- 检查成员是否为字符串中元素：同列表。

```python
s = "hello world"
print("lo" in s)  # True
```

- 原始字符串`r"str"`：没有转义，所见即所得。

```python
print("hello\tworld")  # hello	world
print(r"hello\tworld")  # hello\tworld
```

- 修改与替换：
    - *`str.replace(old, new[, count])`*：返回字符串副本，其中*子字符串`old`*被替换为`new`。
        - `count`：最大替换次数。默认为`-1`，表示替换所有。
    - *`str.split([sep=None][, maxsplit=-1])`*：返回`str`中的单词列表，使用*字符串`sep`*拆分单词。
        - `sep`：默认为`None`，表示任何空白字符串，并从结果中丢弃空字符串。
        - `maxsplit`：最大拆分次数。默认为`-1`，表示不限制。
    - `str.rsplit([sep=None][, maxsplit=-1])`：类似`split()`，从右侧开始拆分。
        - 常用于获取文件路径的后缀或最后一段。
    - *`str.join(iterable)`*：串联`iterable`中的*字符串*。*字符串`str`*作为分隔符。
        - 大厂规范：严禁在循环内使用`+`连接字符串，必须使用`join()`。
- 修剪与移除：
    - `str.strip([chars])`：返回字符串副本，去除前导字符和尾随字符。
        - `chars`：指定要移除的*字符集*。默认为`None`，表示移除空白字符。
    - `str.lstrip([chars])`：类似`strip()`，仅去除前导字符。
    - `str.rstrip([chars])`：类似`strip()`，仅去除尾随字符。
    - `str.removeprefix(prefix)`：如果`str`以`prefix`开头，返回`str[len(prefix):]`；否则返回原字符串副本。
        - Python3.9新增，比`strip()`更安全，不会误删字符集。
    - `str.removesuffix(suffix)`：如果`str`以`suffix`结尾，返回`str[:-len(suffix)]`；否则返回原字符串副本。
        - Python3.9新增。
- 大小写转换：
    - `str.upper()`：返回字符串副本，将每个字符转为大写。
    - `str.lower()`：返回字符串副本，将每个字符转为小写。
    - `str.swapcase()`：返回字符串副本，将每个字符，大写转小写，小写转大写。
    - `str.capitalize()`：返回字符串副本，将第一个字符转为大写，其余转为小写。
    - `str.title()`：返回字符串副本，将每个单词的首字母转为大写（标题格式）。
        - 单词：使用空白字符串分隔。
    - `str.casefold()`：返回字符串副本，将每个字符转为小写。
        - 比`lower()`更彻底（例如德语'ß'会转为'ss'）。
- 查询与统计：
    - *`str.find(sub[, start[, end]])`*：返回*子字符串`sub`*在`str[start:end]`中的最小索引。
        - 如果未找到，则返回`-1`。
    - `str.rfind(sub[, start[, end]])`：返回子字符串`sub`在`str[start:end]`中的最大索引。
        - 如果未找到，则返回`-1`。
    - *`str.index(sub[, start[, end]])`*：类似`find()`。
        - 如果未找到，则引发`ValueError`。
    - `str.rindex(sub[, start[, end]])`：类似`rfind()`。
        - 如果未找到，则引发`ValueError`。
    - `str.count(sub[, start[, end]])`：返回子字符串`sub`在`str[start:end]`中的个数。
    - `str.startswith(prefix[, start[, end]])`：如果`str[start:end]`以指定的`prefix`开头，返回`True`，否则返回`False`。
        - `prefix`：字符串或字符串元组。
    - `str.endswith(suffix[, start[, end]])`：如果`str[start:end]`以指定的`suffix`结尾，返回`True`，否则返回`False`。
        - `suffix`：字符串或字符串元组（检查多个后缀）。
- 格式化与对齐：
    - `str.center(width[, fillchar])`：返回字符串副本，长度为`width`，居中对齐。
        - `fillchar`：填充字符，默认为空格。
    - `str.ljust(width[, fillchar])`：类似`center()`，左对齐。
    - `str.rjust(width[, fillchar])`：类似`center()`，右对齐。
    - `str.zfill(width)`：类似`center()`，右对齐，使用`0`填充。
        - 对正负号有特殊处理，例如：`print("42".zfill(5), "-42".zfill(5))  # "00042" "-0042"`
    - `str.expandtabs(tabsize=8)`：将`str`中的所有`\t`替换为一个或多个空格，具体取决于当前列以及指定的`tabsize`。
- 分割与拆分进阶：
    - `str.splitlines([keepends=False])`：返回`str`中的行列表，使用换行符（`\n`、`\r`、`\r\n`）拆分行。
        - `keepends`：是否保留换行符。
    - `str.partition(sep)`：在`sep`首次出现的位置拆分`str`，返回`(prefix, sep, suffix)`。
        - 如果未找到，返回`(str, "", "")`。
    - `str.rpartition(sep)`：类似`partition()`，从右侧开始拆分。
- 编码与映射：
    - `str.encode(encoding="utf-8"[, errors="strict"])`：返回字符串编码后的字节对象。
        - `errors`：错误处理方案。
    - `str.format_map(dict)`：使用字典等映射关系数据来格式化字符串。 #todo

    - `str.translate(table)`：根据给定的转换表替换或删除字符。
        - 批量替换或删除字符，性能最高的方式。
- 状态判断：
    - `str.isalnum()`：`str`是否非空，且只包含字母（英文字母+汉字）或数字。
    - `str.isalpha()`：`str`是否非空，且只包含字母（英文字母+汉字）
    - `str.isascii()`：`str`是否只包含ASCII字符（空字符串是ASCII字符）。
    - `str.isdecimal()`：`str`是否非空，且只包含十进制数字。
    - `str.isdigit()`：`str`是否非空，且只包含数字（含下标等）。
    - `str.isnumeric()`：`str`是否非空，且只包含数值（含分数、中文数字等）。
    - `str.isidentifier()`：`str`是否为有效的Python标识符。
    - `str.isupper()`：`str`中是否包含至少一个区分大小写的字符，且所有这些（区分大小写的）字符都是大写。
    - `str.islower()`：`str`中是否包含至少一个区分大小写的字符，且所有这些（区分大小写的）字符都是小写。
    - `str.istitle()`：`str`是否非空，且符合title格式（单词首字母大写）。
    - `str.isprintable()`：`str`是否只包含可打印字符（空字符串和空格是可打印字符）。
    - `str.isspace()`：`str`是否非空，且只包含空白字符。
- 内建全局函数：
    - `len(s)`：返回字符串的长度。
    - `max(s)`：返回字符串中Unicode码点最大的字符。
    - `min(s)`：返回字符串中Unicode码点最小的字符。
