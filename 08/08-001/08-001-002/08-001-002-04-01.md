# 字符串 01-02 视频day04 Python文献

- 字符串 (String)：元素有序、元素不可变、元素可重复。
- 定义字符串： #gemini-l
    - 普通字符串：`"str"`（优先）、`'str'`。
    - 文档字符串：`"""str"""`。

```python
# hello 'world'
print("hello 'world'")

"""
hello
world
"""
print("""hello
world""")
```

- 访问：同列表。

```python
s = 'hello world'
print(s[0], s[-1])  # h d
print(s[4:-3])  # o wo
```

- 字符串相加、字符串乘法：同列表。

```python
s1 = "hello"
s2 = "world"
s3 = s1 + s2
print(s3)  # helloworld
print(s3[0] is s1[0], s3[5] is s2[0], s3[4] is s2[1])  # True True True

s4 = s1 * 2
print(s4)  # hellohello
print(s4[0] is s1[0], s4[5] is s1[0])  # True True
```

- 检查成员是否为字符串中元素：同列表。

```python
s = "hello world"
print("lo" in s)  # True
```

- 原始字符串`r"str"`：没有转义，所见即所得。

```python
print("hello\tworld")  # hello	world
print(r"hello\tworld")  # hello\tworld
```

- 常用函数：
    - *`str.replace(old, new[, max])`*：将`str`中的*字符串`old`*替换成`new`。
        - 替换次数 <= `max`
    - *`str.split([sep][, max])`*：返回一个列表。从左往右，将`str`按*字符串`sep`*分隔，分隔后的子字符串作为列表的元素。
        - `sep`默认为空白字符串（`\s+`）。
        - 分隔次数 <= `max`
    - `str.rsplit([sep][, max])`：类似`split()`，从右往左分隔。
    - `sep.join(seq)`：将序列`seq`中所有的字符串，通过分隔符`sep`合并为一个新的字符串。
