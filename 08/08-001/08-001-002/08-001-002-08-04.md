# 自定义异常 10 视频day08 Python文献

- 为什么？
    - 现有的异常类，不匹配当前的场景。
- 怎么做？
    - 自定义一个类（见名知意），直接或间接继承`Exception`。

```python
class MyException(Exception):
    # 无须定义
    # def __init__(self, msg):
    #     self.msg = msg
    #
    # def __str__(self):
    #     return self.msg

    pass


try:
    raise MyException("抛出自定义异常！")
except MyException as e:
    print(e)  # 抛出自定义异常！
```
