# 函数 08-09 视频day04 Python文献

- 多次执行同一项任务 -> 函数：
    - 避免代码冗余。
    - 降低维护成本。
    - 提高可读性。
    - 便于分工协作开发。
- 内建函数（Python提供）、用户自定义函数。
- 定义函数：
    - `return`：`return None`
    - 执行（并非调用）某个函数时，该函数必须提前定义。
        - 模块的代码结构模板： #gemini-l
            1. 导包
            2. 声明全局常量
            3. 定义类
            4. 定义业务逻辑函数
            5. 定义主逻辑函数：`def main():`
            6. 入口检查点：
                - `if __name__ == "__main__":`[^1]
                    - `main()`

```python
def 函数名([形参列表]):
    ["""
        函数说明
    """]
    函数体
    [return [表达式]]
```

[^1]: 只有直接运行这个文件时，才会执行`main()`，避免被导入时立即自动执行。
