# 函数 08-10 视频day04 Python文献

- 函数：封装代码块（实现某个功能），可重复使用。
    - 提高代码的复用性，避免代码冗余。
    - 降低维护成本。
    - 提高可读性。
    - 便于分工协作开发。
- 内建函数（Python提供）、用户自定义函数。
- 定义函数：
    - `形参 = 实参`：
        - 形式参数/形参：在定义函数时，定义者声明的参数。
        - 实际参数/实参：在调用函数时，调用者传递的参数。
    - `return`：等价于`return None`。

```python
def 函数名([形参列表]):
    ["""
        函数说明
    """]
    函数体
    [return [表达式]]
```

- 执行函数：
    - 执行（并非调用）某个函数时，该函数必须提前定义。
        - 模块的代码结构模板： #gemini-l
            1. 导包
            2. 声明全局常量
            3. 定义类
            4. 定义业务逻辑函数
            5. 定义主逻辑函数：`def main():`
            6. 入口检查点：
                - `if __name__ == "__main__":`[^1]
                    - `main()`
    - 内存：
        - 堆内存：存储对象。
            - 模块对象：
                - 模块命名空间`__dict__`：字典[^2]，维护模块的成员。
                    - 模块的成员：变量、常量、函数。
            - 函数对象：
                - 函数命名空间`__dict__`：字典[^2]，维护函数的成员。
                    - 函数的成员：形参、局部变量。
                - 函数体？ #tocheck
        - 栈内存：
            - 函数开始执行，函数入栈。
            - 函数执行结束，函数出栈，释放函数的成员。

def print_star(row, col):

    # i = 1
    # while i <= row:
    while row > 0:
        print("*" * col)

        # i += 1
