- 函数：封装代码块（实现某个功能），可重复使用。
    - 提高代码的复用性，避免代码冗余。
    - 降低维护成本。
    - 提高可读性。
    - 便于分工协作开发。
- 内建函数（Python提供）、用户自定义函数。
- 定义函数：
    - 参数：
        - 在调用函数时，执行`形参 = 实参`。
            - 形式参数/形参：在定义函数时，定义者声明的参数。
            - 实际参数/实参：在调用函数时，调用者传递的参数。
        - 形参形式：
            - 位置参数 (positional argument)：实参根据形参位置匹配形参。
                - 定义：`位置参数`
                    - `形参, ..., /`：`/`前面的形参强制使用位置参数。
                - 调用：`实参`
                    - 实参个数 == 形参个数 and 实参顺序 == 形参顺序
                    - 如果有多个位置参数，可以对容器解包传参：`*容器`。
                        - `*字典`：等价于`*字典.keys()`。
            - 关键字参数 (keyword argument)：实参根据形参名匹配形参。
                - 定义：`关键字参数`（同位置参数）
                    - `*, 形参, ...`：`*`后面的形参强制使用关键字参数。
                - 调用：*`关键字参数=实参`*
                    - 关键字参数之间无所谓顺序。
                    - 如果有多个关键字参数，可以对字典解包传参：`**字典`。
                        - 键名必须和形参名一致。
            - 默认值参数：
                - 定义：*`位置参数=默认值`或`关键字参数=默认值`*
                    - 顺序：*非默认值参数 > 默认值参数*
                - 调用：`实参`
                    - 可省略实参：`默认值`。
                    - 传递实参：`实参`。
            - 可变位置参数 (var-positional argument)：
                - 定义：`*可变位置参数`
                - 调用：`实参1, ...`或`*容器`
                    - 使用元组接收未命名的实参：`(实参1, ...)`。
                    - 可省略实参：`()`。
            - 可变关键字参数 (var-keyword argument)：
                - 定义：`**可变关键字参数`
                    - 顺序：*位置参数 > 可变位置参数 > 关键字参数 > 可变关键字参数*
                - 调用：`参数名1=实参1, ...`或`**字典`
                    - 使用字典接收命名的实参：`{"参数名1": 实参1, ...}`。
                    - 可省略实参：`{}`。
    - `return`：等价于`return None`。

```python
def 函数名([形参列表]):
    ["""
        函数说明
    """]
    函数体
    [return [表达式]]
```

- 执行函数：
    - 执行（并非调用）某个函数时，该函数必须提前定义。
        - 模块的代码结构模板： #gemini-l
            1. 导包
            2. 声明全局常量
            3. 定义类
            4. 定义业务逻辑函数
            5. 定义主逻辑函数：`def main():`
            6. 入口检查点：
                - `if __name__ == "__main__":`[^1]
                    - `main()`
    - 内存模型： #gemini-l
        - 堆内存：存储对象。
            - 模块对象 (`PyModuleObject`)：
                - 命名空间`__dict__` (`dict`[^2])：维护模块的成员。
                    - 模块的成员：全局变量/常量、函数。
            - 函数对象 (`PyFunctionObject`，定义时创建)：
                - `__code__` (`PyCodeObject`)：字节码、变量名元组等。
                - `__globals__`：指向所属模块的`__dict__`。
                - ……
        - 调用栈：存储栈帧。
            - 栈帧 (`PyFrameObject`，开始执行时创建并压栈，执行结束后弹栈并销毁)：
                - `f_code`：指向函数对象的`__code__`。
                - `f_globals`：指向函数对象的`__globals__`。
                - `fastlocals`：数组，存储形参和局部变量的值。 #todo
                - `f_back`：指向调用当前函数的上一个栈帧。
- 在模块中定义函数，在类中定义方法。

```python
def print_star(row, col):
    # 这种方式可以不声明i
    # i = 1
    # while i <= row:
    while row > 0:
        print("*" * col)

        # i += 1
        row -= 1
```
