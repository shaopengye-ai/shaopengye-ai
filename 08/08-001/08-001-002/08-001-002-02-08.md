# 运算符 13 视频day04 08-13 视频day02 Python文献

- 算术运算符（返回结果）：
    - `+`：加、取正。
    - `-`：减、取负。
    - `*`：乘、*多次拼接*。
        - 例如：`s = "--" * 10  # --------------------`
    - `/`：*除*。
        - 例如：`print(10 / 3, type(10 / 3))  # 3.3333333333333335 <class 'float'>`
    - `//`：整除，*商向下取整*。
        - 例如：`print(10 // 3, type(10 // 3))  # 3 <class 'int'>`
    - `%`：*取模*。
        - 例如：`print(10 % 3, -10 % 3, 10 % -3, -10 % -3)  # 1 2 -2 -1`
    - `**`：幂。
        - 例如：`print(2 ** 3)  # 8`
- 赋值运算符（*不返回结果*；首先执行右边的式子）：
    - `=`、`+=`、`-=`、`*=`、`/=`、`//=`、`%=`、`**=`
        - 不支持`++`和`--`，因为数字不可变。
        - 序列支持加法和乘法。
            - 可变（列表）：
                - `c += c1`和`c *= n`：*支持原地修改（优化点）*。
                - `c = c + c1`和`c = c * n`：先执行右边的式子（创建新对象），再赋值。
            - 不可变（字符串、元组）：不支持原地修改。
    - `:=`：赋值并*返回赋的值*。
        - 海象运算符，Python3.8新增。
        - 必须操作返回的值，避免报错。
        - 例如：`print((n1 := 20) < (n2 := 3 ** 2), n1, n2)  # False 20 9`
- 比较运算符（返回结果）：
    - `X == Y`：比较*值或对象的值*。
    - `!=`、`>`、`<`、`>=`、`<=`
    - 数字和字符串不能比较大小。
        - 例如：`print(3.0 > True)  # True`
        - 例如：`print(10 > "3")  # TypeError: '>' not supported between instances of 'int' and 'str'`
    - 字符串和字符串比较大小：按位比较字符编码。
        - 例如：`print("15" > "6")  # False`
- 身份运算符（返回结果）：比较*对象的内存地址*：
    - `X is Y`：`id(X) == id(Y)`，例如：
        - `b = True  # b指向值为1的bool单例对象（全局唯一）。`
        - `i = 1  # i指向小整数池中值为1的int对象。`
        - `print(b == i)  # True`
        - `print(b is i)  # False`
    - `X is not Y`：`id(X) != id(Y)`
- 逻辑运算符（返回结果）：
    - `X and Y`：与。如果`X`为“假”*返回`X`*，否则返回`Y`。
        - 例如：`print(0 and 8, 5 and 8)  # 0 8`
    - `X or Y`：或。如果`X`为“真”*返回`X`*，否则返回`Y`。
        - 例如：`print("" or "hello", "world" or None)  # hello world`
    - `not X`：非。如果`X`为“真”返回`False`；反之返回`True`。
        - 例如：`print(not 0.0, not 3.14)  # True False`
- 位运算符（返回结果）（了解）：
    - `X & Y`：按位与。
    - `X | Y`：按位或。
    - `~ X`：按位取反（包含符号位）。
    - `X ^ Y`：按位异或。
    - `X << N`：按位左移。
    - `X >> N`：按位（有符号）右移。
- 成员运算符（返回结果）：
    - `X in SEQ`：在序列`SEQ`中找到`X`返回`True`，否则返回`False`。
        - 例如：`print(1 in [1, 2, 3, 4, 5])  # True`
    - `X not in SEQ`：在序列`SEQ`中没有找到`X`返回`True`，否则返回`False`。
- 运算符优先级（使用`()`即可）：
    - `**`
    - `~`、`+`（取正）、`-`（取负）
    - `*`、`/`、`//`、`%`
    - `+`（加）、`-`（减）
    - `<<`、`>>`
    - `&`
    - `|`、`^`
    - `>`、`<`、`>=`、`<=`
    - `==`、`!=`、`<>`
    - `=`、`+=`、`-=`、`*=`、`/=`、`//=`、`%=`、`**=`
    - `is`、`is not`
    - `in`、`not in`
    - `and`、`or`、`not`
