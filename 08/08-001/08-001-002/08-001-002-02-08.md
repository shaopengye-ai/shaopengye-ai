# 运算符 08-13 视频day02 Python文献

- 算术运算符：
    - `+`：加。
    - `-`：减、取负。
    - `*`：乘、*多次拼接*。
        - 例如：`s = "--" * 10  # --------------------`
    - `/`：除。
        - 例如：`print(10 / 3, type(10 / 3))  # 3.3333333333333335 <class 'float'>`
    - `//`：整除，*商向下取整*。
        - 例如：`print(10 // 3, type(10 // 3))  # 3 <class 'int'>`
    - `%`：*取模*。
        - 例如：`print(10 % 3, -10 % 3, 10 % -3, -10 % -3)  # 1 2 -2 -1`
    - `**`：幂。
        - 例如：`print(2 ** 3)  # 8`
- 赋值运算符：
    - `=`、`+=`、`-=`、`*=`、`/=`、`//=`、`%=`、`**=`
    - `:=`：赋值并*返回赋的值*。
        - 海象运算符，Python3.8新增。
        - 必须操作返回的值，避免报错。
        - 例如：`print((n1 := 20) < (n2 := 3 ** 2), n1, n2)  # False 20 9`
- 比较运算符：
    - `==`：比较对象的值。
    - `!=`、`>`、`<`、`>=`、`<=`
    - 数字和字符串不能比较大小。
        - 例如：`print(3.0 > True)  # True`
        - 例如：`print(10 > "3")  # TypeError: '>' not supported between instances of 'int' and 'str'`
    - 字符串和字符串比较大小，从最左边开始逐个比较字符串中相应位置的字符的ASCII码。
        - 例如：`print("15" > "6")  # False`
- 逻辑运算符：
    - `X and Y`：与，如果`X`为`False`返回`X`，否则返回`Y`。
    - `X or Y`：或，如果`X`为`True`返回`X`，否则返回`Y`。
    - `not X`：非，如果`X`为`True`返回`False`，反之返回`True`。
