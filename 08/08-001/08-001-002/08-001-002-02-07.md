# 输入和输出 06 视频day02 Python文献

- `input(prompt)`：返回用户输入的字符串，例如：
    - `name = input("请输入您的姓名：")`
    - `print(type(name))  # <class 'str'>`

---

- 普通输出：
    - `print(v)`：打印`v`。
    - `print(v1, v2, ...)`：打印多个`v`，`v`之间用空格分隔。
        - 例如：`print("hello", 1024, True)  # hello 1024 True`
    - `print(v1[, v2, ...], end)`：使用`end`结尾（`end`默认为`"\n"`）。
        - 例如：`print("使用$结尾", end="$")  # 使用$结尾$`
- 格式化输出：
    - ① 在字符串中使用`%`占位（根据顺序传递）。
        - 格式符号：
            - `%o`：将传递的值*转换为*八进制整数。
            - `%d`：将传递的值转换为十进制整数。
            - `%x`：将传递的值转换为十六进制整数。
            - `%[.N]f`：将传递的值转换为浮点数，`.N`表示保留`N`位小数。
            - `%e`：将传递的值转换为科学记数法。
            - `%s`：将传递的值转换为字符串。
        - 例如：`s = "PI: %.3f" % 3.141592657  # PI: 3.142`
        - 例如：`s = "name: %s, age: %d" % ("张三", 23)  # name: 张三, age: 23`
    - ② 在字符串中使用`{}`占位，并调用`format()`。
        - 方式一：根据顺序传递。
            - 例如：`s = "姓名: {}, 性别: {}, 年龄: {}".format("张三", "男", 23)`
        - 方式二：根据位置（基于0）传递。
            - 例如：`s = "姓名: {2}, 性别: {1}, 年龄: {0}".format(23, "男", "张三")`
        - 方式三：根据参数传递。
            - 例如：`s = "姓名: {n}, 性别: {g}, 年龄: {a}".format(a=23, g="男", n="张三")`
        - 使用一对大括号`{}`来转义*一对*大括号。
            - 例如：`s = "{{姓名: {0}, 性别: {1}}}".format("张三", "男")  # {姓名: 张三, 性别: 男}`
    - ③ 在字符串前加`f`，在字符串中的`{}`内引用变量（推荐）。
        - 例如：
            - `name, gender, age = "张三", "男", 23`
            - `s = f"姓名: {name}, 性别: {gender}, 年龄: {age}"`
        - `{变量名 = }`：表示`变量名 = 变量值`。
            - 例如：`s = f"{name = }, {gender = }, {age = }"  # name = '张三', gender = '男', age = 23`
        - 使用一对大括号`{}`来转义*一对*大括号。
            - 例如：`s = f"{{姓名: {name}, 性别: {gender}}}"  # {姓名: 张三, 性别: 男}`
    - 格式化数字（能看懂就行）：
        - `{[位置 | 参数 | 变量名 | 表达式]:[填充符][对齐方式数字宽度][数字分隔符][数字类型]}`
            - ②可以格式化数字，位置和参数在`:`前。
            - ③也可以格式化数字，变量名和表达式在`:`前。
            - 填充符：只能一个字符，默认为空格。
            - 对齐方式：`<`表示左对齐、`^`表示居中对齐，`>`表示右对齐。
            - 数字宽度：宽度不足则使用填充符填充。
            - 数字分隔符：可以使用`,`或`_`，每3位进行分隔。
            - 数字类型：
                - `[#]b`、`[#]o`、`d`、`[#]x`：分别表示将传递的*整数*转换为二进制整数、八进制整数、十进制整数、十六进制整数，`#`表示保留进制的前缀。
                - `[.N]f`、`[.N]%`、`[.N]e`：分别表示将传递的*数字*转换为浮点数、百分比、科学记数法，`.N`表示保留`N`位小数。
        - 例如：`s = "num: {:*^20,.3f}".format(31415.92657)  # num: *****31,415.927*****`
        - 例如：`s = f"姓名: {name}, 年龄: {age:*^6d}"  # 姓名: 张三, 年龄: **23**`
        - 例如：`s = f"n1 = {(n1 := 10):.2f}, n2 = {(n2 := 20):.2f}, (n1 + n2) * 2 = {(n1 + n2) * 2:.2f}"  # n1 = 10.00, n2 = 20.00, (n1 + n2) * 2 = 60.00`
