# 顺序、分支 15-17 视频day02 Python文献

- 流程控制：控制计算机指令的执行顺序。
- 顺序：逐条语句执行。
- 分支：逐个分支判断：
    - 为“假”：先*取反当前分支*（“积累”分支条件），再判断下一分支。
    - 为“真”：先执行分支语句，再*跳出当前结构*（只会执行一个分支）。
- `randint(a, b)`：返回一个`[a, b]`的随机整数。

```python
if 表达式:
    语句
[elif 表达式:
    语句
......]
[else:
    语句]
```

```python
from random import randint

price = 50
balance = randint(0, 100)
print(f"余额为{balance}元。")
if balance < price:
    print("余额不足，请充值。")
else:
    balance -= price
    print(f"消费成功，扣款{price}元，余额为{balance}元。")
print("欢迎下次光临！")
if balance:
    print("余额不为0元。")
else:
    print("余额为0元。")

print("-" * 30)

age = randint(1, 120)
print(f"此人年龄为{age}岁，", end="这是个")
if age < 13:
    print("儿童。")
elif age < 20:  # elif age >= 13 and age < 20:
    print("青少年。")
elif age < 65:  # elif age >= 20 and age < 65:
    print("成年人。")
else:
    print("老年人。")
```
