---

- 分支流程：
    - 逐个分支判断：
        - 为“假”：先*取反当前分支*，再判断下一分支。
            - “积累”分支条件。
        - 为“真”：先执行分支语句，再*跳出当前分支结构*。
            - 一个分支结构只会执行一个分支。
    - 分为：
        - `if`分支
        - `match-case`分支

---

- `if`分支

```python
if 表达式:
    语句
[elif 表达式:
    语句
......]
[else:
    语句]
```

- `match-case`分支
    - 下面的格式仅用于“简单字面量匹配”，还有其他的用法。 #todo

```python
# Python3.10新增
match 对象:
    case 值1[ | 值2 ...]:
        语句
    ......
    case _:  # 匹配一切
        语句
```

- `randint(a, b)`：返回一个`[a, b]`的随机整数。
- 三目运算符：`判断为“真”返回的表达式 if 判断的表达式 else 判断为“假”返回的表达式`

```python
from random import randint

price = 50
balance = randint(0, 100)
print(f"余额为{balance}元。")
if balance < price:
    print("余额不足，请充值。")
else:
    balance -= price
    print(f"消费成功，扣款{price}元，余额为{balance}元。")
print("欢迎下次光临！")
if balance:
    print("余额不为0元。")
else:
    print("余额为0元。")

print("-" * 30)

age = randint(1, 120)
print(f"此人年龄为{age}岁，", end="这是个")
if age < 13:
    print("儿童。")
elif age < 20:  # elif age >= 13 and age < 20:
    print("青少年。")
elif age < 65:  # elif age >= 20 and age < 65:
    print("成年人。")
else:
    print("老年人。")

print("-" * 30)

state = f"0b{randint(0, 1)}{randint(0, 1)}{randint(0, 1)}"
print(f"{state = }")
state = int(state, base=2)

# 不直接获取state的第一位数，而是通过按位与排除其他位数的干扰。
if state & 0b100 == 0b100:
    print("大写状态")
else:
    if state & 0b010 == 0b010:
        if state & 0b001 == 0b001:
            print("简体中文-微软拼音 中文模式")
        else:
            print("简体中文-微软拼音 英文模式")
    else:
        print("英语-美式键盘")

print("-" * 20)

match state:
    case 0b100 | 0b101 | 0b110 | 0b111:
        print("大写状态")
    case 0b011:
        print("简体中文-微软拼音 中文模式")
    case 0b010:
        print("简体中文-微软拼音 英文模式")
    case 0b000 | 0b001:
        print("英语-美式键盘")
    case _:
        print("异常状态码")

print("-" * 30)

num1 = randint(1, 10)
num2 = randint(1, 10)
print(f"{num1 = }, {num2 = }")

if num1 > num2:
    max_num = num1
else:
    max_num = num2
print(f"{max_num = }")

max_num = num1 if num1 > num2 else num2
print(f"{max_num = }")
```
