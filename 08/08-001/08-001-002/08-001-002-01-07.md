# Python的安装 09 视频day01 Python文献

- [ ] 从[Python官网](https://www.python.org/)下载Python安装包。
    - installer：完整版。
        - 包含完整的Python标准库、解释器（CPython）以及一些辅助工具（例如pip）等。
        - cmd执行：
            - `python`：启动解释器。
            - `pip install`：安装第三方库。
    - embeddable package：阉割版。
- [ ] 在cmd执行`python`：
    - [ ] 如果跳转到微软应用商店：
        - [ ] 设置 > 应用 > 高级应用设置 > 应用执行别名
            - [ ] python.exe：`关`
            - [ ] python3.ext：`关`
    - [ ] 如果报错“'python'不是内部或外部命令，也不是可运行的程序或批处理文件”。
        - 未安装Python或Python卸载干净。
- [ ] 安装Python：
    - [ ] Install Python 3.12.8 (64-bit)
        - [ ] Use admin privileges when installing py.exe：`√`
        - [ ] Add python.exe to PATH：`√`（添加环境变量）
        - [ ] `Customize installation`
    - [ ] Advanced Options
        - [ ] Install Python 3.12 for all users：`√`
    - [ ] Setup was successful
        - [ ] `Disable path length limit`（避免路径过长导致的错误）
            - [ ] 注册表编辑器 > `计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\FileSystem` > LongPathsEnabled > 修改
                - [ ] 数值数据：`1`
            - [ ] 重启电脑生效
- [ ] 检验：
    - [ ] 在cmd执行`python --version`，打印`Python 3.12.8`。
