# with关键字 12 视频day08 Python文献

```python
with EXPR[ as VAR]:
    with代码块

# 伪代码
mgr = EXPR
VAR = mgr.__enter__()
exc = True
try:
    with代码块
except:
    exc = False
    if not mgr.__exit__(*sys.exc_info()):
        raise
finally:
    if exc:
        mgr.__exit__(None, None, None)
```

- *`mgr`：上下文管理器*——实现两个魔法方法：
    - `__enter__(self)`：*初始化资源*与准备。
        - 在进入`with`代码块之前，自动调用。
        - `VAR = mgr.__enter__()`
            - *`VAR`：用户操作的对象。*
            - 如果没有`as`，丢弃返回值。
            - 如果enter方法抛出异常：
                - 不会执行`with`代码块。
                - 禁止调用exit方法（避免用新异常掩盖旧异常）。
    - `__exit__(self, exc_type, exc_val, exc_tb)`：*释放资源*（调用close方法）与异常捕获控制。
        - 在退出`with`代码块时（无论是否发生异常），自动调用。
        - `with`代码块正常结束：`mgr.__exit__(None, None, None)`
        - `with`代码块抛出异常：`mgr.__exit__(异常的类型, 异常的值, 异常的回溯信息)`
            - （一般）返回“假”：表示继续向外传递该异常。
            - （严禁盲目）返回“真”：表示`mgr`已经处理了该异常。
