# bool布尔型 02 视频day02 Python文献

- 布尔型只有`True`和`False`，用于真假的判断。
    - 例如：`print(type(True))  # <class 'bool'>`
- 在Python3中，`bool`是`int`的子类，`1`表示`True`，`0`表示`False`，`True`和`False`可以和数字相加。
    - 例如：`print(isinstance(True, int))  # True`
    - 例如：`print(True + 10)  # 11`
    - 例如：`print(False + 10)  # 10`
- 在Python中，一切皆对象。
- 在Python中，如何判断一个对象的“真”与“假”？ #gemini
    - “假”：`False`、零数字（`0`、`0.0`、`0j`）、空字符串、空容器、`None`。
    - “真”：`True`、非零数字、非空字符串、非空容器。
    - 类实例：如果该类定义了`__bool__()`并返回`False`，或者定义了`__len__()`并返回`0`，则为“假”，否则为“真”。
