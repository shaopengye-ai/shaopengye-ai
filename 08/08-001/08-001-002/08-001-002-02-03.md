# bool布尔型 02 视频day02 Python文献

- 布尔型只有`True`和`False`，用于真假的判断。
    - 例如：`print(type(True))  # <class 'bool'>`
- 在Python3中，`bool`是`int`的子类，`1`表示`True`，`0`表示`False`，`True`和`False`可以和数字相加。
    - 例如：`print(isinstance(True, int))  # True`
    - 例如：`print(True + 10)  # 11`
    - 例如：`print(False + 10)  # 10`
- 在Python中，一切皆对象。
- `==`比较对象的值；运算符`is`比较对象的内存地址。例如：
    - `b = True  # 创建一个bool对象，值为1，b指向该对象。`
    - `i = 1  # i指向小整数池中值为1的int对象。`
    - `print(b == i)  # True`
    - `print(b is i)  # False`
- 在Python中，几乎所有对象都赋予了“真”或“假”的属性： #gemini
    - “真”：`True`、非零数字、非空字符串、非空容器、自定义对象。
    - “假”：`False`、零数字（`0`、*`0.0`*）、空字符串、空容器、`None`
