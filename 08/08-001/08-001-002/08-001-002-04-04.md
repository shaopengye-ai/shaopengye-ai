- 字典 (Dictionary)：元素为键值对（根据键访问值），元素无序，元素可变。
    - 键：不可变的数据类型，唯一。
    - 值：任何数据类型，可以重复。
- 创建字典：
    - `dct = {k1: v1, k2: v2, ...}`
        - `{}`、`dict()`：创建空字典。
    - `dct = dict(a1=v1, a2=v2, ...)`
    - `dct = dict(二元组可迭代对象)`
    - `st = {新字典的键: 新字典的值 for 元素 in 可迭代对象 if 表达式 ...}`

```python
dct1 = {}
# <class 'dict'> {}
print(type(dct1), dct1)
dct2 = dict()
# <class 'dict'> {}
print(type(dct2), dct2)

dct3 = {"name": "Alice", "age": 18, "gender": "male"}
# {'name': 'Alice', 'age': 18, 'gender': 'male'}
print(dct3)

dct4 = dict(name="Bob", age=20, gender="female")
# {'name': 'Bob', 'age': 20, 'gender': 'female'}
print(dct4)
# dct5 = dict({("name", "Tom"), ("age", 22), ("gender", "male")})
# dct5 = dict((("name", "Tom"), ("age", 22), ("gender", "male")))
dct5 = dict([("name", "Tom"), ("age", 22), ("gender", "male")])
# {'name': 'Tom', 'age': 22, 'gender': 'male'}
print(dct5)

dct6 = {i: i ** 2 for i in range(5) if i % 2 == 0}
# {0: 0, 2: 4, 4: 16}
print(dct6)
```

- 访问字典（根据键访问值）：
    - `value = dict[key]`
        - `key`不存在时引发`KeyError`。
    - `dict.get(key[, default=None])`：返回`key`对应的值。
        - `key`不存在时返回`default`。
- 向字典中添加元素、修改字典中元素：
    - `dict[key] = value`：`key`不存在时表示添加，`key`存在时表示修改。
- 检查成员是否为字典中的key：
    - `key in dict`
        - 无法直接检查value是否存在。
- 获取字典长度：
    - `len(dict)`
- 遍历字典：
    - `dict.keys()`：返回`dict`键的一个动态视图 (View Object)
        - 字典改变，视图也会跟着变化。
    - `dict.values()`：返回`dict`值的一个动态视图。
    - `dict.items()`：返回`dict`键值对`(key, value)`的一个动态视图。

```python
dct = {"name": "tom", "age": 18, "gender": "male"}

ks = dct.keys()
# <class 'dict_keys'>
print(type(ks))
# name age gender
for k in ks:
    print(k, end=" ")
print()

vs = dct.values()
# <class 'dict_values'>
print(type(vs))
# tom 18 male
for v in vs:
    print(v, end=" ")
print()

kvs = dct.items()
# <class 'dict_items'>
print(type(kvs))
# ('name', 'tom') ('age', 18) ('gender', 'male')
for kv in kvs:
    print(kv, end=" ")
print()
# name:tom age:18 gender:male
for k, v in kvs:
    print(f"{k}:{v}", end=" ")
print()
```

- 删除字典元素：
    - `del dict[key]`：从字典中移除该键值对。
        - 如果`key`不存在，则引出`KeyError`。
