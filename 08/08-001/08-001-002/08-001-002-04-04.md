- 字典 (Dictionary)：字典可变，元素为键值对。
    - 元素无序：
        - Python3.7保持插入顺序。
        - 没有索引 => 无法根据索引访问元素或者切片 -> 根据键访问值。
    - 键：不可变（哈希表），可不同类型，唯一。
    - 值：任意类型，可不同类型，可重复。
- 创建字典：
    - `dct = {k1: v1, k2: v2, ...}`
        - `{}`、`dict()`：创建空字典。
    - `dct = dict(a1=v1, a2=v2, ...)`
    - `dct = dict(字典对象或二元组可迭代对象)`
    - `st = {新字典的键: 新字典的值 *for 元素 in 可迭代对象 *if 表达式}`

```python
dct1 = {}
# <class 'dict'> {}
print(type(dct1), dct1)
dct2 = dict()
# <class 'dict'> {}
print(type(dct2), dct2)

dct3 = {"name": "Alice", "age": 18, "gender": "male"}
# {'name': 'Alice', 'age': 18, 'gender': 'male'}
print(dct3)

dct4 = dict(name="Bob", age=20, gender="female")
# {'name': 'Bob', 'age': 20, 'gender': 'female'}
print(dct4)
# dct5 = dict({("name", "Tom"), ("age", 22), ("gender", "male")})
# dct5 = dict((("name", "Tom"), ("age", 22), ("gender", "male")))
dct5 = dict([("name", "Tom"), ("age", 22), ("gender", "male")])
# {'name': 'Tom', 'age': 22, 'gender': 'male'}
print(dct5)

dct6 = {i: i ** 2 for i in range(5) if i % 2 == 0}
# {0: 0, 2: 4, 4: 16}
print(dct6)
```

- 访问字典：
    - `value = dct[key]`：返回`key`对应的值。
        - 如果`key`不存在，则引发`KeyError`。
    - `dct.get(key[, default=None])`：返回`key`对应的值。
        - 如果`key`不存在，则返回`default`。
- 向字典中添加元素、修改字典中元素：
    - `dct[key] = value`：
        - 如果`key`不存在：添加。
        - 如果`key`存在：修改。
- 检查成员是否为字典中的key：
    - `key in dct`
        - 无法直接检查value是否存在（`value in dct.values()`）。
- 获取字典长度：
    - `len(dct)`
- 遍历字典：
    - `dct.keys()`：返回`dct`键的一个动态视图 (View Object)
        - 字典改变，视图也会跟着变化。
    - `dct.values()`：返回`dct`值的一个动态视图。
    - `dct.items()`：返回`dct`键值对`(key, value)`的一个动态视图。

```python
dct = {"name": "tom", "age": 18, "gender": "male"}

ks = dct.keys()
# <class 'dict_keys'>
print(type(ks))
# name age gender
for k in ks:
    print(k, end=" ")
print()

vs = dct.values()
# <class 'dict_values'>
print(type(vs))
# tom 18 male
for v in vs:
    print(v, end=" ")
print()

kvs = dct.items()
# <class 'dict_items'>
print(type(kvs))
# ('name', 'tom') ('age', 18) ('gender', 'male')
for kv in kvs:
    print(kv, end=" ")
print()
# name:tom age:18 gender:male
for k, v in kvs:
    print(f"{k}:{v}", end=" ")
print()

dct["age"] = 20
dct["address"] = "shenzhen"
# name:tom age:20 gender:male address:shenzhen
for k, v in kvs:
    print(f"{k}:{v}", end=" ")
print()
```

- 删除字典元素：
    - `del dct[key]`：从`dct`中移除指定的键值对。
        - 如果`key`不存在，则引发`KeyError`。
    - `dct.clear()`：就地清空`dct`。
    - `del dct`：删除`dct`。
- 常用函数（了解）：
    - `del dct[key]`：从`dct`中移除指定的键值对。
        - 如果`key`不存在，则引发`KeyError`。
    - `dct.pop(key[, default])`：移除指定的键值对并返回其值。
        - 如果`key`不存在：
            - 设置`default`：返回`default`。
            - 未设置`default`：引发`KeyError`。
    - `dct.popitem()`：移除并返回`dct`中的最后插入的键值对`(key, value)`（遵循LIFO）。
        - 如果`dct`为空，则引发`KeyError`。
    - `dct.clear()`：就地清空`dct`。
    - `dct.update(other)`：使用`other`的键值对更新`dct`。
        - `other`：字典对象或二元组可迭代对象。
    - `dct.get(key[, default=None])`：返回`key`对应的值。
        - 如果`key`不存在，则返回`default`。
    - `dct.setdefault(key[, default=None])`：如果`key`不存在，则插入`key: default`并返回`default`。
        - 如果`key`存在，则返回对应的值。
    - `dct.keys()`：返回`dct`键的一个动态视图 (View Object)
        - 动态：字典改变，视图也会跟着变化。
    - `dct.values()`：返回`dct`值的一个动态视图。
    - `dct.items()`：返回`dct`键值对`(key, value)`的一个动态视图。
    - `dct.copy()`：返回一个浅拷贝`dct`的新字典。
    - `dict.fromkeys(iterable, value=None)`：创建一个新字典，以`iterable`中的元素作为键，`value`作为所有键对应的初始值。
        - 静态方法。
