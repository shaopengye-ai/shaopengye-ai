# 数据类型转换 04 视频day02 Python文献

- 自动类型转换/隐式转换：
    - 两种不同类型的数据进行运算时，结果会提升为较大的数据类型，避免数据丢失。
        - 例如：`print(True + 10, type(True + 10))  # 11 <class 'int'>`
        - 例如：`print(2 + 3.0, type(2 + 3.0))  # 5.0 <class 'float'>`
    - 特殊情况：两个整型进行除法运算，结果是浮点型。
        - 例如：`print(3 / 2, type(3 / 2))  # 1.5 <class 'float'>`
- 字符串只能和字符串拼接`+`。
    - 例如：`print(123 + "456")  # TypeError: unsupported operand type(s) for +: 'int' and 'str'`
- 强制类型转换/显式转换：
    - `int(x[, base])`：将`x`转换为整数，如果`x`为字符串可以使用`base`指定进制。
        - 例如：`print(int("1010", base=2))  # 10`
        - 例如：`print(int("0o12", base=8))  # 10`
        - 例如：`print(int(12.34))  # 12`
    - `float(x)`：将`x`转换为浮点数。
        - 例如：`print(float("12.34"))  # 12.34`
        - 例如：`print(float(12))  # 12.0`
    - `complex(real[, imag])`：创建一个实部为`real`，虚部为`imag`的复数。
        - 例如：`print(complex(3, 5))  # (3+5j)`
    - `str(x)`：将对象`x`转换为字符串。 #todo 魔法方法
    - `repr(x)`：将对象`x`转换为字符串，可以转义字符串中的特殊字符。
    - `eval(x)`：执行字符串`x`表达式，并返回表达式的值。
        - 例如：`eval("print(123)")  # 123`
        - 例如：`eval("print(3 / 2)")  # 1.5`
    - `bin(x)`：将整数`x`转换为二进制字符串。
        - 例如：`print(bin(10), type(bin(10)))  # 0b1010 <class 'str'>`
    - `oct(x)`：将整数`x`转换为八进制字符串。
    - `hex(x)`：将整数`x`转换为十六进制字符串。
    - `ord(x)`：将一个字符`x`转换为对应的ASCII整数值。
        - 例如：`print(ord("A"))  # 65`
        - 例如：`print(ord("a"))  # 97`
    - `chr(x)`：将整数`x`转换为对应的Unicode（万国码/统一码）字符。
        - 例如：`print(chr(65))  # A`
    - `tuple(s)`：将序列`s`转换为一个元组。 #todo
    - `list(s)`：将序列`s`转换为一个列表。
    - `set(s)`：转换`s`为可变集合。
