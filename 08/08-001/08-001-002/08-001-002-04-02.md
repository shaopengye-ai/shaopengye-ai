# 元组 03 视频day04 Python文献

- 元组 (Tuple)：元素有序，元素不可变，元素可重复，元素可不同类型。
    - 使用`()`定义元组，使用`,`分隔元素。
    - 只有一个元素也要有`,`，否则`()`会被当作运算符。

```python
tpl = (0,)
print(tpl)  # (0,)
tpl = (0, 1, 2, 3, 4)
print(tpl)  # (0, 1, 2, 3, 4)
```

- 使用推导式创建生成器：`generator = (新元素 for 元素 in 可迭代对象 if 表达式 ...)`
    - 并非元组推导式。
- 使用生成器创建元组：`tpl = tuple(generator)`

```python
lst = [i for i in range(5)]
print(type(lst))  # <class 'list'>

generator = (i * 2 for i in range(5) if i % 2 == 0)
print(type(generator))  # <class 'generator'>
tpl = tuple(generator)
print(type(tpl), tpl)  # <class 'tuple'> (0, 4, 8)
generator = (i * 2 for i in range(5) if i % 2 == 0)
lst = list(generator)
print(type(lst), lst)  # <class 'list'> [0, 4, 8]

tpl = tuple(i * 2 for i in range(5) if i % 2 == 0)
print(type(tpl), tpl)  # <class 'tuple'> (0, 4, 8)
```

- 访问元组：同列表。

```python
tpl = (0, 1, 2, 3, 4)
print(tpl[2], tpl[-1])  # 2 4
print(tpl[2:4])  # (2, 3)
```

- 元组相加、元组乘法：同列表。

```python
tpl1 = (100, 200, 300)
tpl2 = ("a", "b", "c")
tpl3 = tpl1 + tpl2
print(tpl3)  # (100, 200, 300, 'a', 'b', 'c')
print(tpl3[2] is tpl1[2], tpl3[3] is tpl2[0])  # True True

tpl4 = tpl1 * 2
print(tpl4)  # (100, 200, 300, 100, 200, 300)
print(tpl4[2] is tpl1[2], tpl4[5] is tpl1[2])  # True True
```

- 检查成员是否为元组中元素：
    - `x in tpl`
- 获取元组的元素个数：
    - `len(tpl)`
- 获取元组中的最大值、最小值、所有元素和：
    - `max(tpl)`
    - `min(tpl)`
    - `sum(tpl)`

```python
tpl = (100, 200, 300, 400, 500)

print(100 in tpl)  # True
print(len(tpl))  # 5
print(max(tpl), min(tpl), sum(tpl))  # 500 100 1500
```

- 遍历元组：同列表。

```python
tpl = (100, 200, 300, 400, 500)

# 100 200 300 400 500
for o in tpl:
    print(o, end=" ")
print()

# 0: 100, 1: 200, 2: 300, 3: 400, 4: 500
for i in range(len(tpl)):
    print(f"{i}: {tpl[i]}", end=", " if i < len(tpl) - 1 else "\n")

# 0: 100, 1: 200, 2: 300, 3: 400, 4: 500
for i, o in enumerate(tpl):
    print(f"{i}: {o}", end=", " if i < len(tpl) - 1 else "\n")
```

- 元组的元素不可变

```python
tpl = (100, 200, 300)
tpl1 = tpl
print(tpl1)  # (100, 200, 300)
tpl = tpl + (1, 2, 3)
tpl2 = tpl
print(tpl2)  # (100, 200, 300, 1, 2, 3)
print(f"tpl1 is tpl2: {tpl1 is tpl2}")  # tpl1 is tpl2: False

print("-" * 30)

tpl = (100, 200, 300, [1, 2, 3])
tpl1 = tpl
print(tpl1)  # (100, 200, 300, [1, 2, 3])
tpl[3].append(4)
tpl2 = tpl
print(tpl2)  # (100, 200, 300, [1, 2, 3, 4])
print(f"tpl1 is tpl2: {tpl1 is tpl2}")  # tpl1 is tpl2: True
```
