# float浮点型 01 视频day02 Python文献

- 浮点数：带小数点的数。
    - 例如：`print(type(0.1))  # <class 'float'>`
    - 可以使用科学记数法表示浮点数，例如：`print(1.3e7)  # 13000000.0`
- 在任何编程语言中，浮点型都存在“精度丢失”的问题。
    - 例如：`print(0.1 + 0.2)  # 0.30000000000000004`
    - 解决：导入`decimal`。
        - `from decimal import Decimal`
        - `print(type(Decimal("0.1")))  # <class 'decimal.Decimal'>`
        - `print(Decimal("0.1") + Decimal("0.2"))  # 0.3`
