# PyCharm的安装 10-11 视频day01 Python文献

- [ ] 从[PyCharm官网](https://www.jetbrains.com/pycharm/)下载PyCharm安装包。
- [ ] 安装PyCharm：
    - [ ] 安装选项
        - [ ] 创建桌面快捷方式
            - [ ] PyCharm：`√`
        - [ ] 更新PATH变量（需要重启）
            - [ ] 添加"bin"文件夹到PATH：保持默认`×`[^1]
- [ ] 破解PyCharm： #todo
- [ ] 创建新工程/项目：New Project > Pure Python
    - [ ] Location：`项目的父目录\项目名`
    - [ ] Interpreter type #todo
        - [ ] `Project venv`
            - [ ] Python version：`Python解释器`
            - 虚拟环境(venv)路径：固定为`项目\.venv` => 一个项目，独享一个环境
                - `.venv\Lib\site-packages`：存储第三方库。
                - `.venv\Scripts`：python.exe？
- [ ] 配置PyCharm：
    - [ ] Appearance & Behavior > Appearance
        - [ ] Theme：`Light with Light Header`
    - [ ] Keymap
    - [ ] Editor > Font
        - [ ] Font：`JetBrains Mono`
        - [ ] Size：`20.0`
        - [ ] Line height：`1.2`
        - [ ] Typography Settings
            - [ ] Main weight：`Regular`
            - [ ] Bold weight：`Bold`
            - [ ] Fallback font：`缺省字体`
