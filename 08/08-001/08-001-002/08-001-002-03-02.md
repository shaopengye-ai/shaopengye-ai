# 序列 08 视频day03 Python文献

- 容器：存储若干个元素。 #gemini
    - 数字对象 = 类型 + 值（C的数值/数字化数组） + 引用计数
        - C的数字化数组 => 整数可以无限大
    - 字符串对象 = 类型 + 值（连续的字符序列） + 字符个数 + 引用计数
        - 并非指针数组（指向一个个字符对象 => 内存开销爆炸）。
    - 数字对象和字符串对象的不可变：不可以修改值。
    - 容器对象 = 类型 + 值（指针数组的地址） + 当前元素个数 + 引用计数
        - 可变：可以增删改指针数组的指针 or 可以指向新指针数组（扩容缩容）
        - 不可变：不可以增删改指针数组的指针 and 不可以指向新指针数组
    - 间接修改不可变：创建新对象。
    - 对象的引用计数为0：被垃圾回收器回收。
- 容器这一章有很多函数，可以记不住函数怎么用，但遇到应用场景要知道可以使用什么函数解决。

- 序列 (Sequence)：数据结构：
    - 有序（并非排序）：
        - 遍历元素的顺序，和存储元素的顺序相同。
        - 每个元素都有对应的索引/下标（基于0或-1），可以根据索引来访问和修改元素。
            - 从左往右递增：第一个元素的索引为`0`，第二个元素的索引为`1`，……
            - 从右往左递减：倒数第一个元素的索引为`-1`，倒数第二个元素的索引为`-2`，……
    - 可以存储不同类型的元素。
- 序列的常见类型：`str`、`range`、`list`、`tuple`。
- 序列的通用操作：
    - `sequence[index]`：根据索引访问元素。
    - `sequence[start:stop:step]`：切片。
    - `sequence1 + sequence2`：相加。
    - `sequence * n`：乘法。
    - `x in sequence`：检查成员。
    - `len(sequence)`：获取序列长度。
    - `max(sequence)`：获取序列中元素的最大值。
    - `min(sequence)`：获取序列中元素的最小值。
