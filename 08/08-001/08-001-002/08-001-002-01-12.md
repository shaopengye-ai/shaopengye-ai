- 二进制：0-1，满2进1，以`0b`开头表示。
    - `decimal_number = 0b1010  # 10`
    - `binary_number = bin(10)  # 0b1010`
- 八进制：0-7，满8进1，以`0o`开头表示。
    - `decimal_number = 0o12  # 10`
    - `octal_number = oct(10)  # 0o12`
- 十进制：0-9，满10进1。
- 十六进制：0-F，满16进1，以`0x`开头表示。
    - `decimal_number = 0xa  # 10`
    - `hex_number = hex(10)  # 0xa`
- `x`进制`abc...` -> 十进制`n`：
    - `n = (a * x ** (l - 1)) + (b * x ** (l - 2)) + (c * x ** (l - 3)) + ...`
    - 优化：
        - `n = 0`
        - `n = a + n * x`
        - `n = b + n * x`
        - `n = c + n * x`
        - ……
    - 例如：`0b1011` -> `11`，`0x34a` -> `842`
- 十进制`n` -> `x`进制`m`：
    - 将`n`不断除以`x`，直到商为0为止，然后将每步得到的余数倒过来，即为`m`。
    - 例如：`56` -> `0b111000`，`356` -> `0x164`
- 二进制 -> 八进制/十六进制：
    - 低位开始，将二进制数每三位/四位一组，转换成对应的八进制数/十六进制数。
    - 如果二进制数的位数不是4的倍数，则在高位补0，使其成为4的倍数。
    - 例如：`0x1001011` -> `0x4b`
- 八进制/十六进制 -> 二进制：
    - 将八进制数/十六进制数的每一位，转换成对应的三位/四位的二进制数。
    - 例如：`0x23b` -> `0b1000111011`
