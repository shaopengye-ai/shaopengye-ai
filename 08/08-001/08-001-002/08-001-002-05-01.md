# 浅拷贝和深拷贝 03-04 视频day05 Python文献

- 浅拷贝：
    - 不拷贝（共享）子孙对象。
    - 只拷贝父对象：
        - 不可变对象：不拷贝。
        - 可变对象：拷贝。
- 深拷贝：拷贝父对象及其子孙对象：
    - 不可变对象：
        - 不会被改变：不拷贝。
        - 会被改变（*元组*包含了可变对象）：拷贝。
    - 可变对象：拷贝。
- `import copy`（`copy`是内嵌模块）
    - `copy.copy(x)`：返回一个浅拷贝`x`的新对象。
    - `copy.deepcopy(x)`：返回一个深拷贝`x`的新对象。

```python
import copy


def change_list(lst1):
    print(f"在函数中，修改前，lst1的地址是{id(lst1)}，内容是：")
    print_list(lst1)
    lst1[0] += ("添加的元素",)
    lst1[1].add("添加的元素")
    print(f"在函数中，修改后，lst1的地址是{id(lst1)}，内容是：")
    print_list(lst1)


def print_list(lst):
    for i, item in enumerate(lst):
        print(f"\t索引为{i}的元素：地址是{id(item)}，内容是{item}")


def main():
    lst0 = [(0,), {1}]
    print(f"在函数外，调用前，lst0的地址是{id(lst0)}，内容是：")
    print_list(lst0)

    # lst0_copy = copy.copy(lst0)
    # change_list(lst0_copy)
    lst0_deepcopy = copy.deepcopy(lst0)
    change_list(lst0_deepcopy)

    print(f"在函数外，调用后，lst0的地址是{id(lst0)}，内容是：")
    print_list(lst0)


if __name__ == "__main__":
    main()
```
