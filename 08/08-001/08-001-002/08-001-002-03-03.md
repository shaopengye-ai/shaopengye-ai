- 列表 (List)：可变、有序的元素集合。
    - 使用`[]`定义列表，使用`,`分隔元素，元素的类型可以不同。
    - 元素的索引/下标：
        - 从左往右递增：第一个元素的索引为`0`，第二个元素的索引为`1`，……
        - 从右往左递减：倒数第一个元素的索引为`-1`，倒数第二个元素的索引为`-2`，……
- 访问列表：
    - `lst[index]`：根据索引访问元素。
    - `lst[start:stop:step]`：列表切片（返回一个浅拷贝`lst`的新列表）。
        - `start`：起始索引，包含。
        - `stop`：结束索引，不包含。
        - `step`：步长，默认为`1`。
            - `step > 0`：`start`默认为`0`，`stop`默认为`len(list)`。
            - `step < 0`：`start`默认为`-1`，`stop`默认为`-(len(list) + 1)`。

```python
lst = [100, 200, 300, 400, 500]
print(lst)  # [100, 200, 300, 400, 500]

print(lst[0], lst[-1])  # 100 500

print(lst[:])  # [100, 200, 300, 400, 500]
print(lst[2:4])  # [300, 400]
print(lst[2:])  # [300, 400, 500]
print(lst[:2])  # [100, 200]
print(lst[2:-1])  # [300, 400]
print(lst[::-1])  # [500, 400, 300, 200, 100]
```

- 向列表中添加元素：
    - `lst.append(element)`：在列表末尾追加元素。
    - `lst.insert(index, element)`：在列表指定位置插入元素。

```python
lst.append(600)
print(lst)  # [100, 200, 300, 400, 500, 600]
lst.insert(2, 700)
print(lst)  # [100, 200, 700, 300, 400, 500, 600]
```

- 列表相加：
    - `lst1 + lst2`：返回一个浅拷贝`lst1`和`lst2`的新列表。
- 列表乘法：
    - `lst * N`：返回一个新列表。

```python
lst1 = [100, 200, 300]
lst2 = ["a", "b", "c"]
lst3 = lst1 + lst2
print(lst3)  # [100, 200, 300, 'a', 'b', 'c']
print(lst3[0] is lst1[0], lst3[3] is lst2[0])  # True True

lst4 = lst1 * 2
print(lst4)  # [100, 200, 300, 100, 200, 300]
print(lst4[0] is lst1[0], lst[3] is lst1[0])  # True False
```

- 修改列表中元素：
    - `lst[index] = new_element`
    - `lst[start:stop:step] = [new_element, ...]`：先删除旧元素，再插入新元素。
        - `step == 1 or step == -1`：新元素的个数，可以不等于旧元素的个数（只有一个“缺口”）。
        - `step > 1 or step < -1`：新元素的个数，必须等于旧元素的个数。

```python
lst = [100, 200, 300, 400, 500]
lst[0] = -1
print(lst)  # [-1, 200, 300, 400, 500]
lst[2:4] = ["a", "b", "c"]
print(lst)  # [-1, 200, 'a', 'b', 'c', 500]
# lst[2:5:2] = [2, 3, 4]  # ValueError: attempt to assign sequence of size 3 to extended slice of size 2
lst[2:5:2] = [2, 4]
print(lst)  # [-1, 200, 2, 'b', 4, 500]
```
