# 变量 16-18 视频day01 Python文献

- 声明一个变量并赋值：`变量名 = 变量值`
    - Python是弱类型语言：在声明变量时，无须指定数据类型，但必须赋值。
    - `a = 310`： #gemini-l
        1. *在堆内存中，创建一个对象（`int`类型，值为`310`）。*
        2. *在命名空间（字典）添加一个条目，键为字符串`"a"`，值为该对象的内存地址。*
    - Python的变量是“标签”而非“盒子”。 #gemini-l
        - “盒子”：一个内存空间，存储值。
        - “标签”：*一个对象引用*，指向某个对象。
            - `b = a`：在命名空间添加一个条目，键为`"b"`，值为`a`指向的对象的内存地址。
            - `a = 320`：创建一个新对象（`int`类型，值为`320`），`a`指向新对象。
- 声明多个变量并赋值：
    - 例如：`v1 = v2 = v3 = 10`
    - 例如：`v4, v5, v6 = 10, 20, 30`
- 交换两个变量的值：*`v1, v2 = v2, v1`*
    - 底层使用元组来实现。
- 标识符：可以自定义命名。
    - 例如：变量名、函数名、类名、源文件名/模块名……
    - 命名规则：
        - 只能包含字母（区分大小写）、数字和下划线，且不能以数字开头。
        - 不能和关键字重名。
    - 模块的命名可以不遵循命名规则，但不建议。
    - 命名规范： #gemini-l
        - 变量名、函数名、模块名：单词全小写，连接下划线。
        - 类名：单词首字母大写。
        - 常量名：单词全大写，连接下划线。
            - 常量的声明和变量相同，但没有强约束性[^1]。
        - 私有变量、私有函数：开头下划线（表示不建议外部直接访问）。

[^1]: Python没有严格意义上的常量，仅是约定其值不会改变。
