# 面向对象的三大特性之一：多态 13 视频day07 Python文献

- 多态：一个对象，作为方法的参数或返回值时，有多种不同形态。
    - [[08-001-003-02|多态的定义 Python闪念]]
- 如何实现多态？ #gemini-l
    - “不要通过继承来谋求多态”——在Python中，继承主要用于代码复用。
    - 使用“鸭子类型”来实现多态。
        - 鸭子类型 (Duck Typing)：不检查对象的类型，而是检查是否存在特定的方法。
            - “如果它走起来像鸭子，叫起来也像鸭子，那么它就是鸭子。”
                - 一个函数能不能接受某个对象，不取决于这个对象“是谁的子类”，而取决于这个对象“有没有对应的方法”。
            - 灵活性极高：无须继承或包装，只须要有特定的方法。
        - 使用`Protocol`来显式声明“鸭子类型”的协议（接口契约）。

```python
from typing import Protocol


# 定义一个协议，不需要被继承。
class Animal(Protocol):

    def do(self) -> None:
        ...


# 显式不继承Animal，解耦彻底。
class Bird:

    def __init__(self, age):
        self.age = age

    def do(self) -> None:
        print(f"一只{self.age}岁的小鸟正在飞...")


class Pig:

    def __init__(self, name):
        self.name = name

    def do(self) -> None:
        print(f"一只名为{self.name}的小猪正在跑...")


# 静态类型检查器：确保animal必须实现了do方法。
def do(animal: Animal) -> None:
    animal.do()


def new_animal(factory: lambda: Animal) -> Animal:
    return factory()


b = new_animal(lambda: Bird(3))
do(b)
p = new_animal(lambda: Pig("xx"))
do(p)

# do(o)
```
