# 字符的编码和解码 05 视频day02 Python文献

- 编码表：字符和二进制的映射表。
    - 编码和解码必须使用相同的编码表，避免乱码。
- 编码：字符串`str` -> 字节序列`bytes`，例如：
    - `str1 = "你好"`
    - `bs = str1.encode(encoding="utf-8")  # 不指定encoding默认使用utf-8。`
    - `print(bs, type(bs))  # b'\xe4\xbd\xa0\xe5\xa5\xbd' <class 'bytes'>`
- 解码：字节序列`bytes` -> 字符串`str`，例如：
    - `str2 = bs.decode(encoding="utf-8")  # 不指定encoding默认使用utf-8。`
    - `print(str2, type(str2))  # 你好 <class 'str'>`
