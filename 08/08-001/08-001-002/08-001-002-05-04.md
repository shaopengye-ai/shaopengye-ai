- 匿名函数/lambda表达式，对比函数：
    - 没有函数名。
        - *临时使用*，不可重复使用。
    - 函数体：等价于`return 表达式`。
        - 只能封装*有限的逻辑*。
        - 只能访问自己的形参。
- 面向函数式编程或匿名函数的应用：将操作逻辑和函数解耦。

```python
"""
方式一
    操作逻辑和函数的耦合度大。
        => 如果增删改操作逻辑，需要修改函数的代码。
"""
def calculate1(n1, n2, operator):
    if operator == "+":
        return n1 + n2
    elif operator == "-":
        return n1 - n2


print(calculate1(1, 2, "+"))


"""
方式二
    将操作逻辑和函数解耦。
"""
def calculate2(n1, n2, operator):
    return operator(n1, n2)


def add(n1, n2):
    return n1 + n2


print(calculate2(1, 2, add))

print(calculate2(1, 2, lambda a, b: a + b))
```

- `sorted(iterable, /, *, key=None, reverse=False)`：返回一个新列表，`iterable`按照比较键排序。
    - `key`：函数对象，接收一个元素，从该元素提取比较键，返回比较键。
        - 默认为`None`，表示直接比较元素。
    - `reverse`：默认为`False`，表示升序。
- `map(func, *iterables)`：返回一个`map`对象，同时操作若干个可迭代对象的每个元素的结果。
    - `func`：函数对象，接收若干个可迭代对象的元素，操作这些元素，返回操作结果。
    - 最短的可迭代对象决定遍历的终点。
- `filter(function, iterable)`：返回一个`filter`对象，可迭代对象中判断为“真”的元素。
    - `function`：函数对象，接收一个元素，判断该元素，返回“真”或“假”。
        - 如果为`None`，等价于`lambda e: e`。
- `reduce(function, iterable[, initializer])`：返回归约/聚合一个可迭代对象的结果。
    - `from functools import reduce`
    - `function`：函数对象，接收两个参数，聚合这两个参数，返回聚合结果。
    - `initializer`：初始值。
        - 如果没有初始值，`function`一开始接收第一个元素和第二个元素，将聚合后的结果作为下一次调用的第一个参数，另一个参数则为第三个元素……
        - 如果有初始值，`function`一开始接收初始值和第一个元素，将聚合后的结果作为下一次调用的第一个参数，另一个参数则为第二个元素……

```python

lst = range(5)
```
