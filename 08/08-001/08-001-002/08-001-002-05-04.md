# 匿名函数 14-15 视频day05 Python文献

```python
lambda 形参列表: 表达式
```

- 匿名函数，对比函数：
    - 没有函数名。
    - 函数体：等价于`return 表达式`。
        - 只能封装有限的逻辑。
        - 只能访问自己的形参。

```python
# 方式一
def calculate1(n1, n2, operator):
    if operator == "+":
        return n1 + n2
    elif operator == "-":
        return n1 - n2


print(calculate1(1, 2, "+"))


# 方式二
def calculate2(n1, n2, operator):
    return operator(n1, n2)


def add(n1, n2):
    return n1 + n2


print(calculate2(1, 2, add))


# 方式三
def calculate3(n1, n2, operator):
    return operator(n1, n2)


print(calculate3(1, 2, lambda a, b: a + b))
```
