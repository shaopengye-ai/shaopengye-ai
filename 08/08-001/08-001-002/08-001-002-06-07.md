# 类的定义 10 视频day06 Python文献

```python
class 类名:
    ["""
    类说明文档
    """]

    类的成员
```

- 类名：单词首字母大写。
- 类说明文档：`__doc__`
- 类的成员：
    - 类属性/类变量：
        - 定义：
            - *在类内方法外：`类属性 = 值`*
            - 在方法内、在类外：`类对象.新的类属性 = 值`
        - 访问：*`类对象.类属性`、`类对象.类属性 = 新值`。*
            - `实例对象.未覆盖的类属性`：读取类属性。
                - 如果该实例对象存在和该类属性同名的实例属性，则优先读取实例属性。
            - `实例对象.属性 = 值`：创建或修改实例属性。
        - 所有实例对象**共享类对象的命名空间**的类属性。
    - 实例属性/实例变量：
        - 定义：
            - *在init方法内：`self.实例属性 = 值`*
            - 在其他方法内：`self.新的实例属性 = 值`
            - 在类外：`实例对象.新的实例属性 = 值` 
        - 访问：*`实例对象.实例属性`、`实例对象.实例属性 = 新值`。*
        - 每个实例对象**独享自己的命名空间**的实例属性。
            - 有什么实例属性？
            - 有什么实例属性值？
    - 实例方法：
        - 定义：在类内：`def 实例方法(self):`
            - 第一个参数固定为`self`（约定俗成的参数名），表示实例对象本身。
        - 调用：
            - 方式一：`类对象.实例方法(实例对象)`
            - 方式二：`实例对象.实例方法()`（底层实现为方式一）
        - 可以访问：
            - `self` -> 未覆盖的类属性、实例属性、类方法、实例方法。
            - 类对象 -> 类属性、类方法、实例方法。
    - 类方法：
        - 定义：在类内：使用`@classmethod`装饰器装饰，`def 类方法(cls):`
            - 第一个参数固定为`cls`（约定俗成的参数名），表示类对象本身。
        - 调用：
            - 方式一：`类对象.类方法()`
                - 无须创建实例对象，一般用于获取类的成员。
            - 方式二：`实例对象.类方法()`
        - 可以访问：类对象 -> 类属性、类方法。
- 类的操作：
    - 成员引用：`类名.类属性`、`类名.实例方法`……
    - 实例化：`对象名 = 类名([属性值, ...])`
        - 过程：
            1. `__new__(cls, *args, **kwargs)`：创建对象。
                - 构造方法：创建对象：`return object.__new__(cls)`
                - `cls`：类对象。
            2. `__init__(self)`：初始化对象：添加实例属性并赋值。
                - 如果没有显式地提供init方法，Python会提供默认的init方法（不执行任何操作）。
                - init方法只能返回`None`，否则引发`TypeError`。
        - `对象名.类属性`、`对象名.实例属性`、`对象名.实例方法`、`对象名.实例方法()`。
- `self`：
    - *必须通过`self`来调用实例属性和实例方法。*
