# 类的定义 10-18 视频day06 Python文献

```python
class 类名:
    ["""
    类说明文档
    """]

    类的成员
```

- 类名：单词首字母大写。
- 类说明文档：`__doc__`
- 类的成员：
    - 类属性/类变量：
        - 定义：在类内，方法外：`类属性 = 值`
        - 访问：`类对象.类属性`、`类对象.类属性 = 新值`。
            - `实例对象.未覆盖的类属性`：读取类属性。
                - 如果该实例对象存在和该类属性同名的实例属性，则优先读取实例属性。
            - `实例对象.属性 = 值`：创建或修改实例属性。
        - 所有实例对象**共享类对象的命名空间**的类属性。
    - 实例属性/实例变量：
        - 定义：在init方法内：`self.实例属性 = 值`
        - 访问：`实例对象.实例属性`、`实例对象.实例属性 = 新值`。
        - 每个实例对象**独享自己的命名空间**的实例属性。
            - 有什么实例属性？
            - 有什么实例属性值？
    - 静态方法：
        - 定义：在类内：使用`@staticmethod`装饰器装饰，`def 静态方法():`
            - 不依赖类属性或实例属性。
        - 调用：`类对象.静态方法()`
            - 无须创建实例对象，一般用于定义工具类的方法。
            - 也可以`实例对象.静态方法()`。
        - 可以访问：类对象 -> 类属性、静态方法、类方法。
    - 类方法：
        - 定义：在类内：使用`@classmethod`装饰器装饰，`def 类方法(cls):`
            - 第一个参数固定为`cls`（约定俗成的参数名），表示类对象本身。
        - 调用：`类对象.类方法()`
            - 无须创建实例对象，一般用于获取类的成员。
            - 也可以`实例对象.类方法()`。
        - 可以访问：`cls` -> 类属性、静态方法、类方法。
    - 实例方法：
        - 定义：在类内：`def 实例方法(self):`
            - 第一个参数固定为`self`（约定俗成的参数名），表示实例对象本身。
        - 调用：`实例对象.实例方法()`
            - 底层实现：`类对象.实例方法(实例对象)`
        - 可以访问：
            - `self` -> 实例属性、实例方法。
            - 类对象 -> 类属性、静态方法、类方法。
- 实例化：`对象名 = 类名()`
    - 过程：
        1. `__new__(cls, *args, **kwargs)`：创建对象。
            - 构造方法：创建对象：`return object.__new__(cls)`
            - `cls`：类对象。
        2. `__init__(self)`：初始化对象：添加实例属性并赋值。
            - 如果没有显式地提供init方法，Python会提供默认的init方法（不执行任何操作）。
            - init方法只能返回`None`，否则引发`TypeError`。
- 魔法方法：在执行特定操作时，Python自动调用。
    - 命名规范：`__魔法方法__()`
    - 常见的魔法方法：
        - new方法
        - init方法
        - del方法：定义当对象被垃圾回收时（引用计数为0）的行为。
            - `del 变量名`：减少某个对象的引用计数。
        - str方法：定义`str(实例对象)`的行为：产生人类可读的输出，类似Java的`toString()`。
        - repr方法：定义`repr(实例对象)`的行为：产生机器可读的输出，一般为有效的Python代码。
        - getattribute方法：属性访问拦截器，定义属性被访问前的行为。
