# 抛出异常 09 视频day08 Python文献

- `raise 异常类("异常的描述信息")`：创建并抛出对应的异常类对象。
    - 在满足某些条件时（例如：检查函数的实参），明确表示发生了异常，可以使用`raise`抛出异常，停止程序的正常执行，并将控制权转移到异常处理部分。
- `assert 表达式[, "异常的描述信息"]`：断言。
    - 等价于`if not 表达式:    raise AssertionError(["异常的描述信息"])`。
        - 只能抛出`AssertionError`。

```python
def add1(x, y):
    if not isinstance(x, int) or not isinstance(y, int):
        raise TypeError("参数类型错误！")

    return x + y


def add2(x, y):
    assert isinstance(x, int) and isinstance(y, int), "参数类型错误！"

    return x + y


# print(add1(1, 2))  # 3
# print(add1("1", 2))  # TypeError: 参数类型错误！
print(add2(1, 2))  # 3
print(add2("1", 2))  # AssertionError: 参数类型错误！
```

- 程序正常执行完毕，但输出结果不符合预期，此时需要调试程序：
    - 跟踪变量或对象的变化：`print(var)`
    - 断言：`assert 表达式[, "异常的描述信息"]`
    - debug
