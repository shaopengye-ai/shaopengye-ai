# 集合 04-05 视频day04 Python文献

- 集合 (Set)：元素无序，元素可变，元素不可重复，元素可不同类型。
    - 无序 => 无法根据索引或者切片访问元素
    - 应用：数学上的集合操作（并集、交集和差集）、快速成员检查、消除重复项和集合运算。
- 创建集合：
    - `st = {o1, o2, ...}`
    - `st = set(iterable)`
        - 创建空集合：`set()`
        - 创建空字典：`{}`
    - `st = {新集合的元素 for 元素 in 可迭代对象 if 表达式 ...}`

```python
st1 = {1, 2, 3}
print(type(st1), st1)  # <class 'set'> {1, 2, 3}
st2 = set(range(1, 4))
print(type(st2), st2)  # <class 'set'> {1, 2, 3}

st3 = set()
print(type(st3), st3)  # <class 'set'> set()

st4 = {i * 2 for i in range(5) if i % 2 == 0}
print(type(st4), st4)  # <class 'set'> {0, 8, 4}
```

- 向集合中添加元素：
    - `set.add(element)`：向集合中添加一个元素。
        - `element`：必须是可哈希（hashable）的对象。
        - 如果元素已存在，则不进行任何操作。

```python
st = set(range(5))
print(st)  # {0, 1, 2, 3, 4}
st.add(5)
print(st)  # {0, 1, 2, 3, 4, 5}
st.add(5)
print(st)  # {0, 1, 2, 3, 4, 5}
```

- 从集合中删除元素：
    - `set.remove(element)`：从集合中移除指定的元素。
        - 如果元素不存在，则引发`KeyError`。

```python
st = set(range(5))
print(st)  # {0, 1, 2, 3, 4}
st.remove(2)
print(st)  # {0, 1, 3, 4}
# st.remove(2)
# print(st)  # KeyError: 2
```

- 检查成员是否为集合中元素：
    - `x in st`
- 获取集合的元素个数：
    - `len(st)`
- 获取集合中的最大值、最小值、所有元素和：
    - `max(st)`
    - `min(st)`
    - `sum(st)`
- 遍历集合：
    - `for o in st:`
