# 集合 04-05 视频day04 Python文献

- 集合 (Set)：集合可变，元素不可变，元素可不同类型，元素唯一，元素无序。
    - 无序 => 没有索引 => 无法根据索引访问元素或者切片。
    - 应用：快速成员检查、去重和集合运算。
- 创建集合：
    - `st = {o1, o2, ...}`
        - `{}`：创建空字典。
    - `st = set(iterable)`
        - `set()`：创建空集合。
    - `st = {新集合的元素 *for 元素 in 可迭代对象 *if 表达式}`

```python
st1 = {1, 2, 3}
print(type(st1), st1)  # <class 'set'> {1, 2, 3}
st2 = set(range(1, 4))
print(type(st2), st2)  # <class 'set'> {1, 2, 3}

st3 = set()
print(type(st3), st3)  # <class 'set'> set()

st4 = {i * 2 for i in range(5) if i % 2 == 0}
print(type(st4), st4)  # <class 'set'> {0, 8, 4}
```

- 向集合中添加元素：
    - `st.add(element)`：向集合中添加一个元素。
        - 如果元素已存在，则不进行任何操作。

```python
st = set(range(5))
print(st)  # {0, 1, 2, 3, 4}
st.add(5)
print(st)  # {0, 1, 2, 3, 4, 5}
st.add(5)
print(st)  # {0, 1, 2, 3, 4, 5}
```

- 从集合中删除元素：
    - `st.remove(element)`：从集合中移除指定的元素。
        - 如果元素不存在，则引发`KeyError`。

```python
st = set(range(5))
print(st)  # {0, 1, 2, 3, 4}
st.remove(2)
print(st)  # {0, 1, 3, 4}
# st.remove(2)  # KeyError: 2
```

- 检查成员是否为集合中元素：
    - `x in st`
- 获取集合的元素个数：
    - `len(st)`
- 获取集合中的最大值、最小值、所有元素和：
    - `max(st)`
    - `min(st)`
    - `sum(st)`
- 遍历集合：
    - `for o in st:`
- 常用函数：
    - `st.add(element)`：向`st`中添加一个元素。
        - `element`：必须是可哈希（hashable）的对象。
        - 如果元素已存在，则不进行任何操作。
    - `st.remove(element)`：从`st`中移除指定的元素。
        - 如果元素不存在，则引发`KeyError`。
    - `st.discard(element)`：从`st`中移除指定的元素。
        - 如果元素不存在，也不引发错误。
    - `st.pop()`：从`st`中移除并返回一个任意元素。
        - 无序 => 无法保证移除的是哪一个。
        - 如果`set`为空，则引发`KeyError`。
    - `st.clear()`：就地清空`st`。
    - `st.update(*iterable)`：修改`st`，`st`与`iterable`的并集。
        - `*iterable`：一个或多个可迭代对象。
    - `st.union(*iterable)`：返回一个新集合，`st`与`iterable`的并集。
    - `st.intersection_update(*iterable)`：修改`st`，`st`与`iterable`的交集。
    - `st.intersection(*iterable)`：返回一个新集合，`st`与`iterable`的交集。
    - `st.difference_update(*iterable)`：修改`st`，`st`与`iterable`的差集。
    - `st.difference(*iterable)`：返回一个新集合，`st`与`iterable`的差集。
    - `st.symmetric_difference_update(iterable)`：修改`st`，`st`与`iterable`的对称差集。
        - 对称差集：在`st`或`iterable`中，但不同时存在于两者的元素。
    - `st.symmetric_difference(iterable)`：返回一个新集合，`st`与`iterable`的对称差集。
    - `set3 = set1 | set2`：返回一个新集合，`set1`和`set2`的并集。
        - `|`：并集运算符，要求两侧均为`set`对象。
    - `set3 = set1 & set2`：返回一个新集合，`set1`和`set2`的交集。
        - `&`：交集运算符，要求两侧均为`set`对象。
    - `set3 = set1 - set2`：返回一个新集合，`set1`和`set2`的差集。
        - `-`：差集运算符，要求两侧均为`set`对象。
        - 在`set1`中但不在`set2`中的元素。
    - `st.isdisjoint(iterable)`：如果`st`与`iterable`没有交集，则返回`True`。
    - `st.issubset(iterable)`：如果`st`是`iterable`的子集，则返回`True`。
    - `st.issuperset(iterable)`：如果`st`是`iterable`的超集，则返回`True`。
    - `st.copy()`：返回一个浅拷贝`st`的新集合。
    - `len(st)`：返回集合元素个数。
    - `max(st)`：求集合中元素的最大值。
    - `min(st)`：求集合中元素的最小值。
    - `sum(st)`：求集合中元素的加和。
