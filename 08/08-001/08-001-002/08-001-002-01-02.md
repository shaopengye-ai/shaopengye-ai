# 计算机的组成 01 视频day01 Python文献

- 计算机 = 硬件 + 软件
    - 计算机主要用于高速计算。
    - 人形机器人也是计算机。
- *硬件 = 输入设备 + 存储器 + 中央处理器 + 输出设备*
    - 存储器 (Memory)
    - 中央处理器 (Central Processing Unit, CPU)：*中央处理器 = 运算器 + 控制器*
        - 运算器 = 算术逻辑单元 + 累加器 + 状态寄存器 + 通用寄存器
            - *算术逻辑单元* (ALU)：具体的算术运算和逻辑运算。
        - *控制器* (Control Unit)： #gemini-l
            1. 从内存中*取指令*，并*译码*（例如是进行算术运算还是存数据等等）。
            2. 根据译码结果，向其他部件（例如算术逻辑单元、存储器等等）*发出控制信号*。
            3. 确保各部件按照严格的时间顺序*协调工作*。
- 软件 = 系统软件 + 应用软件
