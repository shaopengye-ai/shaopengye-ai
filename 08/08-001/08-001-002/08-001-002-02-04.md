# String字符串初识 03 视频day02 Python文献

- 字符串：字符序列（羊肉串）。
    - 例如：`print(type('hello'))  # <class 'str'>`
- 在Python中，用引号括起来的都是字符串：
    - 一个单引号/双引号。
        - 例如：`print("hello 'world'")  # hello 'world'`
    - 三个单引号/双引号：原样保留字符串（*换行生效*）。
        - 例如：某一行`print("""hello`，下一行`world""")`
- `\`：
    - 行尾：表示续行。
    - 其余：转义特殊字符。
        - 例如：`\\`、`\'`、`\"`、`\t`、`\n`、`\r`。
- intern滞留机制：声明字符串（要求没有空格或特殊符号，但PyCharm无视）时，首先检查内存中是否存在相同的字符串，如果有则不创建，直接指向该字符串。
    - 例如：`print(id('helloworld') == id('helloworld'))  # True`
