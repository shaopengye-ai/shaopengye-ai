# 变量是“标签”而非“盒子”

- “盒子”：一个内存空间，存储值。
- “标签”：*一个对象引用*，存储某个对象的内存地址（指向该对象）。
    - => 在Python中，一切皆对象。
    - => 在[[08-001-03-02|声明变量]]时，无须指定数据类型[^1]，但必须赋值。
    - => `变量2 = 变量1`：`变量2`和`变量1`指向同一个对象。

```python
# 1. 在堆内存中，创建一个值为310的int对象。
# 2. 在命名空间（字典）添加一个条目，键为字符串"a"，值为该对象的内存地址。
a = 310

# 在命名空间添加一个条目，键为"b"，值为a指向的对象的内存地址。
b = a

# 在堆内存中，创建一个值为320的int对象，a指向该对象。
a = 320

print(a is b)  # False
```

```python
# 1. 在堆内存中，创建一个值为指针数组的list对象。
# 2. 在命名空间添加一个条目，键为"c"，值为该对象的内存地址。
c = [0, 1, 2]

# 在命名空间添加一个条目，键为"d"，值为c指向的对象的内存地址。
d = c

# 该对象的指针数组的索引为1的指针，指向值为10的int对象。
d[1] = 10

print(c is d, c)  # True [0, 10, 2]
```

[^1]: 变量只是一个对象引用，指向某个对象，因此变量没有类型，对象才有类型。
