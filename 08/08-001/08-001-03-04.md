# 变量是“标签”而非“盒子”

- “盒子”：一个内存空间，存储值。
- “标签”：*一个对象引用*，存储某个对象的内存地址（指向该对象）。
    - => 在Python中，一切皆对象。
    - => 在[[08-001-03-02|声明变量]]时，无须指定数据类型[^1]，但必须赋值。

```python
# 1. 在堆内存中，创建一个值为`310`的`int`对象。
# 2. 在命名空间（字典）添加一个条目，键为字符串`"a"`，值为该对象的内存地址。
a = 310

# 在命名空间添加一个条目，键为`"b"`，值为`a`指向的对象的内存地址。
b = a

# 在堆内存中，创建一个值为`320`的`int`对象，`a`指向该对象。
a = 320
```
