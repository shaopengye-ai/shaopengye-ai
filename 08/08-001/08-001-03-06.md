# 进制转换

- `x`进制`abc...` -> 十进制`n`：
    - `n = (a * x ** (l - 1)) + (b * x ** (l - 2)) + (c * x ** (l - 3)) + ...`
    - 优化：
        - `n = 0`
        - `n = a + n * x`
        - `n = b + n * x`
        - `n = c + n * x`
        - ……
    - 例如：`0b1011` -> `11`，`0x34a` -> `842`
- 十进制`n` -> `x`进制`m`：
    - 将`n`不断除以`x`，直到商为`0`为止，然后将每步得到的余数倒过来，即为`m`。
    - 例如：`56` -> `0b111000`，`356` -> `0x164`
- 二进制 -> 八进制/十六进制：
    - 从最低位开始，将二进制数每三位/四位一组，转换成对应的八进制数/十六进制数。
    - 如果二进制数的位数不是三/四的倍数，则在高位补`0`，使其成为三/四的倍数。
    - 例如：`0b1001011` -> `0x4b`
- 八进制/十六进制 -> 二进制：
    - 将八进制数/十六进制数的每一位，转换成对应的三位/四位的二进制数。
    - 例如：`0x23b` -> `0b1000111011`
