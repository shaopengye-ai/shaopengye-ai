# 进制转换

- [`base`进制`m` -> 十进制`n`](08-001-03-06-01.py.md)
- [十进制`n` -> `base`进制`m`](08-001-03-06-02.py.md)
- 二进制 -> 八进制/十六进制：
    - 从最低位开始，将二进制数每三位/四位一组[^1]，转换成对应的八进制数/十六进制数。
    - 例如：`0b1001011` -> `0x4b`
- 八进制/十六进制 -> 二进制：
    - 将八进制数/十六进制数的每一位，转换成对应的三位/四位的二进制数。
    - 例如：`0x23b` -> `0b1000111011`

[^1]: 如果二进制数的位数不是三/四的倍数，则在高位补`0`，使其成为三/四的倍数。
